## Reporting a Vulnerability

Security is a top priority for this project. If you discover a security vulnerability, please follow the steps below to report it responsibly.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to report
1. **Email:** Please report vulnerabilities by emailing **uncannyrishabh@gmail.com** or open a new issue.
2. **Details:** Include as much detail as possible, including:
   - A description of the vulnerability.
   - Steps to reproduce the issue (PoC).
   - The potential impact of the exploit.

* **Disclosure:** Once the vulnerability is fixed, we will coordinate a public disclosure if necessary, giving you proper credit for the discovery unless you wish to remain anonymous.
